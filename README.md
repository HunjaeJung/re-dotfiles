# re-dotfiles

스스로 이해한 dotfiles만 사용해보기로 했습니다. 

쓸데없어 보이고 괜히 힘들수있지만, 적어도 지금은 그게 가치있는 작업일 것 같다는 생각을 합니다. 

# 설치

```
git clone --recursive https://github.com/HunjaeJung/re-dotfiles.git ~/.dotfiles
cd ~/.dotfiles
python install.py
```
